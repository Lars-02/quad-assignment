# Open Trivia Challenge - Frontend

This is the frontend user interface for the Open Trivia Challenge. It allows users to answer trivia questions, submit
their answers safely to a custom backend, and receive detailed feedback on their scores.

## Technologies Used

* **Framework:** React with Vite
* **Language:** TypeScript
* **Styling:** Tailwind CSS & HeroUI
* **Data Fetching:** SWR
* **Validation:** Zod

## Prerequisites

Before you begin, ensure you have [Node.js](https://nodejs.org/) and npm installed on your system.

## Getting Started

Install Dependencies:
```sh
  npm install
```

Start development mode:
```sh
  npm run dev
```
