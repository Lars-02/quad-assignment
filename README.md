# Open Trivia Challenge - Project

This is the full-stack application built for the assignment that tests your general knowledge using some trivia, built
with a Java Spring Boot backend and using React in the frontend.

## How to Run

This project uses a decoupled architecture. **You must run the backend and the frontend separately in two different
terminal windows** for the application to function correctly.

The frontend relies on the backend to fetch the questions securely and calculate your final score.

In both the frontend and backend folders are README files that have more context about their respective parts.
